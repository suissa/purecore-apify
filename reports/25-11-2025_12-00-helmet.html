🎯 Funcionalidades do HelmetGuard

🔧 Configuração Flexível

Habilite/desabilite headers individualmente com opções customizáveis para cada política de segurança.

HelmetGuard({ contentSecurityPolicy: { ... }, strictTransportSecurity: { maxAge: 31536000 }, referrerPolicy: { policy: 'strict-origin-when-cross-origin' } })

🎨 CSP Avançado

Content Security Policy completo com suporte a todas as diretivas modernas e modo report-only.

@CSPGuard({ directives: { 'default-src': ["'self'"], 'script-src': ["'self'", 'https://cdn.example.com'], 'report-uri': '/csp-report' }, reportOnly: false })

🔒 HSTS Robusto

Strict Transport Security com preload, subdomains e configuração de tempo de vida customizável.

@HSTSGuard({ maxAge: 63072000, // 2 anos includeSubDomains: true, preload: true })

🌐 Cross-Origin Policies

COEP, COOP e CORP para isolamento completo de origens cruzadas e proteção avançada.

@COEPGuard({ policy: 'require-corp' }) @COOPGuard({ policy: 'same-origin' }) @CORPGuard({ policy: 'same-origin' })

🔄 Middleware Compatível

Função helmet() compatível com Express para uso como middleware tradicional.

// Como middleware app.use(helmet({ contentSecurityPolicy: false }));

⚙️ Configuração via Ambiente

Todas as opções configuráveis via variáveis de ambiente para diferentes ambientes.

ENABLE_CSP=true HSTS_MAX_AGE=31536000 REFERRER_POLICY=strict-origin-when-cross-origin

📝 Como Usar

1. HelmetGuard Completo (Recomendado)

import { HelmetGuard } from '@purecore-br/4pi'; class SecureController { @HelmetGuard() // Todos os headers ativados async secureEndpoint(req, res) { res.json({ security: 'maximum' }); } }

2. Configuração Personalizada

class ApiController { @HelmetGuard({ contentSecurityPolicy: { directives: { 'default-src': ["'self'"], 'script-src': ["'self'", "'unsafe-inline'"], 'style-src': ["'self'", 'https:', "'unsafe-inline'"] } }, strictTransportSecurity: { maxAge: 31536000, includeSubDomains: true, preload: false }, referrerPolicy: { policy: 'strict-origin-when-cross-origin' }, xFrameOptions: { action: 'DENY' } }) async customSecurity(req, res) { res.json({ customized: true }); } }

3. Decorators Individuais

import { CSPGuard, HSTSGuard, XFrameOptionsGuard, ReferrerPolicyGuard } from '@purecore-br/4pi'; class FlexibleController { @CSPGuard({ directives: { 'default-src': ["'self'"], 'img-src': ["'self'", 'data:', 'https:'] } }) @HSTSGuard({ maxAge: 63072000, preload: true }) @XFrameOptionsGuard({ action: 'DENY' }) @ReferrerPolicyGuard({ policy: 'no-referrer' }) async flexibleSecurity(req, res) { res.json({ flexible: true }); } }

4. Como Middleware Express

import { helmet } from '@purecore-br/4pi'; const app = express(); // Helmet completo app.use(helmet()); // Configuração personalizada app.use(helmet({ contentSecurityPolicy: false, // Desabilitar CSP strictTransportSecurity: { maxAge: 31536000 } }));
