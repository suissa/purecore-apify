# 🚀 Como Executar o AutoGeneratorDDD

## ❌ Problema Identificado

Os scripts não estão sendo executados no terminal atual. Isso pode ser devido a:

1. **Configuração do PowerShell/Windows Terminal**
2. **Problemas com ES Modules no Node.js**
3. **Falta do Bun runtime**
4. **Configurações de segurança**

## ✅ Soluções para Executar

### 🔥 SOLUÇÃO 1: Usar Node.js com arquivo temporário

**Passo 1:** Abra um novo terminal/cmd e navegue até o projeto:
```bash
cd D:\www\Freelas\PROJETOS_OPENSOURCE\MEUS\@purecore\purecore-apify
```

**Passo 2:** Crie um arquivo temporário `temp-run.mjs`:
```javascript
import { autoGenerateDDDFromZodSchemas } from './src/auto-generator-ddd.js';

console.log('🎯 Executando AutoGeneratorDDD...');

await autoGenerateDDDFromZodSchemas({
  modulesPath: 'examples',
  domainPath: 'src/domain',
  boundedContext: 'user-example',
  verbose: true,
  force: true,
  dryRun: true  // ← MUDE PARA false PARA GERAR ARQUIVOS REAIS
});

console.log('✅ Concluído!');
```

**Passo 3:** Execute o arquivo:
```bash
node temp-run.mjs
```

**Passo 4:** Limpe o arquivo temporário:
```bash
del temp-run.mjs
```

### 🎯 SOLUÇÃO 2: Usar Browser Console (Mais fácil!)

**Passo 1:** Abra o arquivo `examples/manual-execution.ts`

**Passo 2:** Copie o código da função `executarGeradorSimples()`:

```typescript
await autoGenerateDDDFromZodSchemas({
  modulesPath: 'examples',
  domainPath: 'src/domain',
  boundedContext: 'user-example',
  verbose: true,
  force: true,
  dryRun: true

```

**Passo 3:** Abra o browser e vá para uma página em branco

**Passo 4:** Abra o Developer Tools (F12) e vá para Console

**Passo 5:** Cole e execute o código copiado

---

### 🛠️ SOLUÇÃO 3: Usar TypeScript diretamente

**Passo 1:** Verifique se o TypeScript compila:
```bash
npx tsc --noEmit src/auto-generator-ddd.ts
```

**Passo 2:** Se compilar sem erros, execute:
```bash
npx ts-node-esm examples/run-ddd-generator.ts
```

---

### 📝 SOLUÇÃO 4: Modificar package.json (Recomendado)

**Passo 1:** Edite o `package.json` e adicione:
```json
{
  "scripts": {
    "ddd:generate": "node --loader ts-node/esm examples/run-ddd-generator.ts"
  }
}
```

**Passo 2:** Instale ts-node se necessário:
```bash
npm install -D ts-node
```

**Passo 3:** Execute:
```bash
npm run ddd:generate
```

## 🐛 Troubleshooting

### ❌ "Cannot find module" ou "Module not found"
- Verifique se `examples/user.schema.ts` existe
- Verifique se tem a exportação `export const schema = UserSchema;`

### ❌ "TypeError" ou "SyntaxError"
- Execute: `npx tsc --noEmit src/auto-generator-ddd.ts`
- Corrija os erros de TypeScript

### ❌ Nenhum output/console log
- Verifique se está no diretório correto
- Tente usar caminhos absolutos

### ❌ Erro de permissão
- Execute como administrador
- Ou use uma pasta sem restrições
